# Getting Started

## Installation

1. Download **ScheduleLink** from the [Autodesk App Store](https://apps.autodesk.com/)
2. Close all Revit instances
3. Run the installer
4. Launch Revit — the **ScheduleLink** button will appear in the **Add-Ins** tab

## First Use

### Step 1: Open a Schedule

Open any schedule view in Revit. ScheduleLink works with all schedule types:

- Room Schedules
- Sheet Lists
- Door / Window Schedules
- MEP Equipment Schedules
- Material Takeoff Schedules
- Any custom schedule

### Step 2: Launch ScheduleLink

Go to **Add-Ins** tab → Click **ScheduleLink**

The main dialog opens showing:
- **Left panel** — List of all schedules in the project
- **Right panel** — Parameter preview for the selected schedule
- **Search box** — Filter schedules by name
- **Export / Import buttons** — At the bottom

### Step 3: Select a Schedule

Click on a schedule in the left panel. The right panel shows all parameters with color coding:

| Color | Meaning |
|-------|---------|
| 🟢 Green | Editable parameter — can be modified via Import |
| 🔴 Red/Pink | Read-only parameter — exported for reference only |
| 🟡 Yellow | Calculated parameter — cannot be modified |

### Step 4: Export or Import

- **Export** — Saves the selected schedule to an Excel file
- **Import** — Reads an Excel file and updates the Revit elements

See [Export Guide](export-guide.md) and [Import Guide](import-guide.md) for detailed instructions.

## Uninstallation

1. Go to **Control Panel** → **Programs and Features**
2. Find **ScheduleLink** and click **Uninstall**
3. Restart Revit

Alternatively, remove via the Autodesk App Store desktop app.
